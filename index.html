Sir Tim Berners-Lee

The Man Who Invented the World Wide Web

Timeline

  • 1955 - Born in London, England.
  • 1976 - Graduates from The Queen's College, Oxford with a degree in physics.
  • 1980 - First proposes the concept of hypertext and builds the ENQUIRE prototype while working at CERN.
  • 1984 - Returns to CERN after working at John Poole's Image Computer Systems, Ltd.
  • 1989 - Proposes the concept of the World Wide Web - an internet-based hypertext information system.
  • 1990 - Develops core World Wide Web tools, including the HyperText Transfer Protocol, HyperText Markup Language, the WorldWideWeb web browser, and the CERN httpd web server.
  • 1991 - Launches the first public website at info.cern.ch.
  • 1994 - Founds the World Wide Web Consortium (W3C).
  • 2004 - Knighted by Queen Elizabeth II for "services to the global development of the internet."
